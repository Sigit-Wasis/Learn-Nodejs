Apa itu Node js?

Node.js, atau singkatnya Node, adalah lingkungan runtime JavaScript sisi server yang open source. Node.js memungkinkan JavaScript untuk digunakan di banyak tempat di luar browser, seperti di sisi server misalnya. Ada beberapa fitur penting yang ditawarkan oleh Node.js, yang membuatnya sangat menarik.

Node.js adalah pembungkus di sekitar mesin JavaScript yang disebut V8. V8 mendukung banyak browser, termasuk Google Chrome, Opera, dan Microsoft Edge. Node.js memungkinkan Anda untuk mengeksekusi JavaScript dengan mesin V8 di luar browser. Selain itu, ini berisi banyak optimasi untuk V8 bahwa aplikasi yang berjalan di server mungkin perlu. Misalnya, Node.js menambahkan sesuatu yang disebut "Buffer" ke V8. "Buffer" ini memungkinkan V8 bekerja dengan file. Ini membuatnya menjadi pilihan yang luar biasa untuk membangun sesuatu seperti Server Web.

Jika Anda belum pernah menggunakan JavaScript sebagai bahasa pemrograman utama sebelumnya, Anda mungkin berpikir, "apakah ini benar-benar ide yang bagus?" JavaScript adalah pilihan tepat sebagai bahasa pemrograman utama! Anda tidak hanya dapat menulis aplikasi modular yang tangguh dengan JavaScript, tetapi ada keuntungan lain yang hanya dimiliki JavaScript. Misalnya, karena browser juga menggunakan JavaScript, Node.js memungkinkan untuk berbagi logika seperti aturan validasi formulir antara browser dan server.

JavaScript lebih relevan saat ini karena munculnya Aplikasi Halaman Tunggal (SPA). Selain itu, JSON (Notasi Objek JavaScript) adalah format pertukaran data yang banyak digunakan dan JavaScript memiliki dukungan asli untuknya. Banyak teknologi basis data NoSQL seperti CouchDB dan MongoDB menggunakan JavaScript dan JSON sebagai format kueri dan skema. Menggunakan Node.js berarti Anda sudah menggunakan bahasa dan teknologi yang sama dengan yang digunakan banyak layanan dan kerangka kerja modern.

Aplikasi yang dapat Anda bangun dengan Node.js:

  • HTTP Web Servers.
  • Microservices or Serverless API backends.
  • Drivers for database access and querying.
  • Interactive command-line interfaces.
  • Desktop applications.
  • Real-time IoT client and server libraries.
  • Plugins for Desktop applications.
  • Shell scripts for file manipulations or network access, etc.
  • Machine learning libraries and models.

Ekosistem Node.js juga kaya berkat NPM, manajer paket untuk Node.js di mana setiap orang dapat berbagi perpustakaan Node.js mereka sendiri. Juga, bayangkan Anda ingin membangun perangkat untuk dapat mengirim perintah kontrol ke meja Anda; bukankah itu keren? Node.js akan sangat cocok untuk kasus penggunaan ini juga. Node.js cepat dan performan dan dapat menangani aplikasi real-time dan aliran data berat. Anda akan menginstal Node.js di papan IoT Anda, atau menggunakan perangkat yang dilengkapi dengan Node.js yang sudah diinstal sebelumnya. Lalu tulis logika aplikasi Anda dalam JavaScript dan gunakan di perangkat, itu saja.

Di bagian selanjutnya, kita akan menarik penutup Node.js dan melihat arsitekturnya, dan cara kerjanya.
