Introduction

Tailwind Traders adalah perusahaan ritel besar dengan banyak data, banyak file dan banyak masalah untuk dipecahkan. Anda telah direkrut oleh Tailwind, dan Anda telah mengetahui mereka ingin melakukan segalanya dengan Node.js. Tapi Anda tidak tahu cara menggunakan Node.js - Tidak masalah! Cara terbaik untuk belajar adalah "di tempat kerja". Dalam modul ini, Anda akan mulai dari awal, mempelajari apa sebenarnya Node.js dan dari mana asalnya. Bagaimana akhirnya kami menulis JavaScript di sisi server, dan apa yang dapat Anda lakukan dengan itu? Pada akhir modul ini, Anda akan dapat memutuskan apakah Node.js adalah alat yang tepat untuk proyek Anda di Tailwind Traders.

Tujuan Pembelajaran

Pada akhir modul ini, Anda akan dapat:

  • Jelaskan apa itu Node js
  • Jelaskan cara kerja Node.js
  • Identifikasi kapan harus menggunakan Node.js
  • Buat dan jalankan skrip Node.js dari baris perintah

Prasyarat

  • Keakraban dengan mengedit teks dan file kode dalam editor teks apa pun
  • Pengalaman tingkat pemula menulis kode JavaScript
  • Pengalaman menggunakan baris perintah
